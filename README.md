🎓 College Discovery Platform

Find the best engineering institutes in Maharashtra.

🌟 Project Purpose
Choosing the right college is a major decision for students. This platform was created to simplify that journey by providing a fast, clean, and intuitive way to explore engineering institutes.

This project was developed as the core requirement for Task B of the Full-Stack Web Development Internship, focusing on bridging the gap between a database and a user-friendly interface.

✨ Key User Features
Instant Search: Find colleges immediately by typing their name (e.g., "MIT" or "COEP").

Professional Card Layout: High-quality visual cards that display the college name, location, and annual fees at a glance.

Real-Time Data: The platform is directly connected to a cloud database, ensuring the information you see is live and accurate.

Responsive Design: Works perfectly whether you are searching on a mobile phone, tablet, or laptop.

🛠️ The Power Behind the Platform
While the user sees a simple interface, the platform uses a modern "Power Trio" of technologies:

Next.js: Provides the lightning-fast speed of the website.

Supabase: A secure cloud database that stores all college records.

Tailwind CSS: Used to create the professional, modern "look and feel" of the platform.
